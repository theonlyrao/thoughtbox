##To run tests
1. clone down and cd into the directory
2. `bundle`
3. `rake db:create db:migrate`
4. `rspec`

##To run locally

1. clone down and cd into the directory
2. `bundle`
3. `rake db:create db:migrate`
4. `rails s`
